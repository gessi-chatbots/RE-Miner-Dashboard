RE-Miner

The main goal of RE-Miner is to provide a user-friendly, easily accessible and reusable software tool for both researchers and mobile app developers. It is designed to facilitate replication studies and comparative analyses in the field of review-based natural language processing tasks, with a particular emphasis on feature extraction and emotion classification.

The tool is composed of two software-based contributions:

  1. RE-Miner-Hub, a web-based service supporting the integration and comparison of feature extraction and emotion classification tasks, using various NLP models and providers. The source code of this software component is available at https://github.com/gessi-chatbots/RE-Miner-Hub.
  2. RE-Miner-Dashboard, a cloud-based application to support user management, app and review persistence, automatic processing and analytical visualization of a batch of user reviews combining features and emotions emerged from these reviews. The source code of this software component is available at https://github.com/gessi-chatbots/RE-Miner-Dashboard.
